# Code of Conduct

Ormica adopts the **[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its Code of Conduct.

In short: we want Ormica to be a welcoming, collaborative space for everyone — regardless of background, identity, or experience level. Be kind. Be respectful. Assume good intent. Help others learn.

## Scope

This Code of Conduct applies in all project spaces:

- GitHub repository (issues, pull requests, discussions, code review)
- Any future Discord / Slack / mailing list
- Public communication about the project on behalf of the maintainers

## Reporting

If you observe or experience behaviour that conflicts with this Code, please report it privately to the project maintainer:

> **ormica.maintainer@gmail.com**

Reports are handled confidentially. The maintainer will respond as quickly as possible and discuss next steps with the reporter.

## Enforcement

Maintainers may take any action they consider appropriate, including warning, temporary or permanent ban from the project's spaces, or escalation to platform-level moderation (GitHub abuse reports, etc.).

For the full text of expectations, enforcement guidelines, and the four-tier response model, please read the canonical Contributor Covenant document:

📜 **https://www.contributor-covenant.org/version/2/1/code_of_conduct/**
